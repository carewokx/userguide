## **Création/Adhésion/Sortie/Suppression d'équipes**

### **🔹 Création et Adhésion à une équipe**

Vous pouvez facilement créer et rejoindre une équipe. Cependant, vous ne pouvez être affilié qu'à une seule équipe à la fois.

- **Création d'une équipe**

  - Une équipe est créée en saisissant simplement le nom de l'équipe et votre nom.

  - Vous pouvez inviter des collègues juste après la création pour l'utiliser ensemble.

- **Adhésion à une équipe**

  - Vous pouvez adhérer instantanément en saisissant le **code d'identification**, le **mot de passe** et le nom que vous souhaitez utiliser.

  - Vous pouvez également rejoindre une équipe en acceptant une invitation d'un membre existant.

  ***

### **🔹 Invitation de membres d'équipe**

Vous pouvez rejoindre une équipe par invitation même si vous ne connaissez pas le code d'identification ou le mot de passe.

**👥 Inviter depuis la liste d'amis (Valable 24 heures)**

- Seuls les amis pouvant être invités s'affichent dans la liste.  
  **🔑 Inviter avec un code d'identification personnel (Valable 24 heures)**
- Chaque utilisateur possède un code d'identification unique à 8 chiffres.
- Votre code d'identification personnel peut être vérifié sous votre nom dans l'onglet **[Plus]**.  
  **🔗 Partager le lien d'invitation**
- Vous pouvez inviter des amis en partageant le lien via messagerie ou réseaux sociaux.

❓ **Si « L'utilisateur ne peut pas être invité » s'affiche**

- L'utilisateur n'existe pas (code d'identification incorrect).
- L'utilisateur a désactivé l'application.
- L'utilisateur est déjà affilié à une autre équipe.
- L'utilisateur a déjà reçu une invitation.

  ***

### **🔹 Comment quitter une équipe**

**Membre général**

- **Départ immédiat** : **[Lieu de travail > Infos équipe > Paramètres > Quitter l'équipe]**

- **Départ programmé** : Définissez la date de fin de **service** dans **[Lieu de travail > Infos équipe > Paramètres > Modifier mes infos]** (départ automatique à minuit ce jour-là).

**Administrateur**

- Si vous avez des privilèges d'administrateur, vous ne pouvez pas quitter l'équipe directement.

- Vous ne pouvez partir qu'une fois qu'un autre administrateur a supprimé vos privilèges d'administrateur.

- Si l'administrateur est le dernier membre restant, la seule façon de partir est de supprimer l'équipe.

  ***

### **🗑️ Suppression d'équipe et réinitialisation du mot de passe**

Les équipes qui ne sont plus utilisées peuvent être supprimées.

- Le mot de passe défini lors de la création est requis pour la suppression. Utilisez **Réinitialiser** si vous ne connaissez pas le mot de passe.

⚠️ Lorsqu'une équipe est supprimée, **tous les membres sont automatiquement retirés**. Veuillez décider avec soin.

---

### **📌 Foire aux questions**

**Q. Où puis-je consulter les équipes dans lesquelles j'ai travaillé auparavant ?**

- Dans **[Lieu de travail > Infos équipe]**, vous pouvez consulter la liste des équipes actuelles et passées, leurs codes d'identification et leurs périodes d'activité.

**Q. Que se passe-t-il si je supprime une équipe de la liste des anciennes équipes ?**

- Votre historique de travail restant dans le planning de cette équipe disparaît, et le **Type de service** pour cette période dans votre planning personnel est remplacé par le type que vous avez défini personnellement.
