## **Congé annuel · Demi-journée de congé : Gestion facile des congés**

Vous pouvez consulter l'état de vos congés en un coup d'œil et gérer facilement tout, de l'enregistrement des congés annuels/demi-journée au suivi des jours restants.

---

### **⚙️ Paramètres de congé**

Veuillez saisir le nombre total de jours de congé que vous utiliserez cette année dans le menu **[Statut des congés > Paramètre des congés]**.  
 Les informations saisies sont automatiquement reflétées comme suit :

- **Les congés annuels/demi-journées de congé enregistrés sur le planning de service sont automatiquement déduits.**

- Si vous avez déjà utilisé des congés, vous pouvez les saisir ensemble pour **calculer les jours restants avec précision.**

- Vous pouvez ensuite **modifier ou supprimer les paramètres** dans le menu **[Gestion des congés > Modifier]**.

---

### **🗓️ Congé annuel/demi-journée enregistré sur le planning de travail**

  Lorsque vous enregistrez un congé annuel ou une demi-journée de congé sur votre planning de **service**, il est **automatiquement déduit** du nombre total de jours de congé que vous avez défini.

- **Congé annuel (R)** : 1 jour déduit

- **Demi-journée (½)** : 0,5 jour déduit

- Les jours de **Repos (OFF)** n'affectent pas le nombre de jours de congé.

  ⚠️ Si vous n'enregistrez pas le congé sur votre planning de **service**, il ne sera pas déduit de vos jours de congé restants. N'oubliez pas de l'enregistrer.

---

### **⏰ Comment saisir une demi-journée de congé**

  La demi-journée de congé peut être utilisée **conjointement avec d'autres types de service.**
  Cependant, vous devez d'abord avoir un **service** saisi pour cette date spécifique pour enregistrer une demi-journée de congé.
  Par exemple, pour saisir **‘DAY + Demi-journée’ le 21 mai** :

1. Dans **[Modifier le service]**, commencez par **saisir le service DAY** pour cette date.

2. Ensuite, sélectionnez Demi-journée (½), et elle sera enregistrée au format **‘DAY ½’**.

   ***

### **📊 Onglet Statut des congés**
Dans l'onglet **[Statut des congés]**, vous pouvez vérifier vos **jours de congé restants en fonction du paramètre de congé le plus récent** que vous avez saisi.

Si vous modifiez les paramètres de congé, les informations affichées sont automatiquement reflétées.  
  - 📌 Notez que **le statut des congés est calculé sur la base de tous les congés saisis dans le planning de service, quel que soit le mois actuellement visible à l'écran.**  
  Par exemple, même si vous visualisez le mois de mai, le calcul des jours restants inclut les congés utilisés jusqu'en juillet s'ils ont été saisis.
